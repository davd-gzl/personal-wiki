# Data

Curated datasets (mostly geographic) and IPFS-pinned resources.

## Format

- **GeoJSON** — Curated POIs (cafés, coworking, etc.)
- **IPFS links** — Large or persistent datasets

## IPFS Links
*Empty*

### How to access

```bash
# Gateway
https://ipfs.io/ipfs/<CID>

# Local node
ipfs cat <CID>
```

**Pinning services:** [Pinata](https://pinata.cloud) · [Web3.Storage](https://web3.storage)
